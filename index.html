Paper Sculpture This project combines modern aesthetics with elements of Chinese culture, resulting in a striking sleek silver and black geometrical dragon-shaped helmet. Crafted using paper and aluminum foil, the piece symbolizes strength and tradition.
Clay Modeling Inspired by Malaysian product packaging, this clay sculpture takes the form of a playful cow. A milk box is incorporated into the material to emphasize the theme. This project highlights the blend of functional design and artistic expression.
Color Harmony This project explores the concept of color harmony using a portrait manipulated with Adobe Photoshop's “Cut-Out” effect. Two compositions were created using analog colors and complementary colors. The result demonstrates an understanding of balanced color relationships.
My Inner Creature This creative project began with experimenting with spilled coffee stains, uncovering intriguing shapes and forms. Using an Artline Pen, these abstract shapes were transformed into imaginative creatures.
Reflective Blended Drawing This project uses the Reductive Drawing Technique to capture the complexity of reflective and bent surfaces. The exercise demonstrates an understanding of light, shadow, and perspective in creating dynamic, blended compositions.
Printmaking Exploring the art of printmaking, this project involved creating collagraph. Using materials such as lino ink, mounting board and woodcut tools, a fire rose motif was designed. This project showcases craftsmanship and creativity in traditional printmaking methods.
Digital Manipulation This project explores the art of digital collage to create a visually captivating portrait inspired by the fairy core theme. Using various textures, images, and elements, the final piece showcases intricate layering techniques and a strong sense of composition, bringing the magical world of fairy tales to life.
Line Art Tapestry This project experiments with the expressive qualities of lines using unconventional tools such as forks, tissues, brushes, and toothpicks on a drawing block. The result demonstrates a diverse range of textures, patterns, and movements, emphasizing the versatility and creativity of linework in artistic expression.
3D Mythical Creature This project explores digital modeling through 3D Builder software, bringing the beloved BT21 characters Chimmy and Tata to life. The models demonstrate experimentation with digital design tools and precision in character creation.
Type As Image This project delves into creative typography, beginning with 26 experimental sketches of the alphabet. The final products include Arson, a bold and fiery digital typography design. The work showcases a fusion of artistic imagination and technical execution in communication design.
